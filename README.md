# vue-ui-docs

## 介绍

VueCli+Markdown+Travis 自定解析组件库 Markdown 示例，自动发布到 Npm 和 github pages。

## Vue 项目创建

1. 安装 Vue CLI

```bash
npm install -g @vue/cli
```

2. 启动 vue ui 可视化操作创建项目

```bash
vue ui
```
